# 운동 플래너 (healthapp)

헬스(무산소)와 유산소 운동을 날짜별로 계획하고 수행하는 iOS(Cupertino) 스타일 Flutter 앱.

## 기능

- **날짜별 계획**: 주간 캘린더에서 날짜를 선택해 그 날의 운동을 계획. 아래 화살표로 월 전체 보기로 확장.
  계획이 있는 날은 파란 점, 모두 완료한 날은 초록색으로 표시.
- **무산소 운동**: 운동 이름, 세트 수, 세트별 반복 횟수·무게(kg)·휴식 시간을 설정.
  수행 화면에서 세트를 완료하면 해당 세트의 휴식 타이머가 자동 시작 (+30초 연장 / 건너뛰기, 종료 시 알림음·진동).
- **유산소 운동**: 이름과 지속 시간을 설정. 수행 화면에서 카운트다운 타이머(시작/일시정지/재설정) 제공, 시간이 다 되면 자동 완료 처리.
- **루틴 복사·공유**: 하루/1주일 루틴을 복사해 다른 날짜·주에 붙여넣기.
  복사 시 `FIT1.` 공유 코드가 클립보드에 저장되어 다른 사람에게 전달 가능 ("공유 코드 가져오기"로 가져옴).
- **성장 그래프**: 운동별 기록 추이를 라인 차트로 확인 — 무산소는 최대 무게/총 볼륨/총 횟수, 유산소는 지속 시간.
- **설정**: 타이머 진동/알림음 on·off, 한 주의 시작 요일(일/월), 모든 데이터 삭제.
- **진행률 표시**: 날짜별 완료 현황 프로그레스 바, 완료 체크 표시.
- **영구 저장**: shared_preferences 로 로컬 저장 (앱 재시작 후에도 유지).

## 실행

```sh
flutter pub get
flutter run          # 연결된 기기/에뮬레이터
flutter run -d web-server --web-port=8473   # 웹 브라우저
```

## 구조

- `lib/models.dart` — 운동/세트 모델, JSON 직렬화, 날짜·시간 포맷 헬퍼
- `lib/storage.dart` — 날짜별 계획 저장소 (shared_preferences)
- `lib/settings.dart` — 앱 설정 저장소 + 설정 화면
- `lib/routine_share.dart` — 루틴 복사 버퍼와 공유 코드 인코딩/디코딩
- `lib/main.dart` — 홈 화면 (캘린더 + 일일 운동 목록 + 루틴 복사/붙여넣기)
- `lib/edit_screens.dart` — 무산소/유산소 추가·편집 화면
- `lib/detail_screens.dart` — 수행 화면 (세트 체크 + 휴식 타이머, 유산소 타이머)
- `lib/stats_screens.dart` — 운동별 성장 그래프 (커스텀 라인 차트)
- `lib/widgets.dart` — 공용 Cupertino 스타일 위젯
